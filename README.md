## 🚀 프로젝트 소개

### 위치 정보를 활용한 **약속 관리 앱**

👬 만나고 싶은 친구들을 초대해보세요

📅 캘린더에서 내 약속을 관리할 수 있어요

📍 약속장소에 오기로 한 친구들이 어디쯤에 있는지 확인할 수 있어요

⏰ 약속시간이 다가오면 알림을 받을 수 있어요
